# CoviHeal

CoviHeal is a **Proof of Concept (POC)** healthcare platform designed to assist users during the COVID-19 pandemic. It acts as a one-stop solution for essential resources, providing real-time hospital bed availability, oxygen supply tracking, and live updates on the pandemic. CoviHeal is focused on streamlining healthcare support and delivering accurate, timely information to users in critical situations.

## Key Features

### 1. Real-Time Hospital Resource Tracking
- Integrates APIs from government and healthcare organizations to ensure accurate and up-to-date information.

### 2. Oxygen Supply Locator
- Enables users to search for oxygen suppliers in their vicinity.
- Includes a verification system to ensure supplier authenticity.

### 3. Live COVID-19 Statistics
- Displays real-time updates on COVID-19 cases, recoveries, and fatalities.
- Offers regional and national-level data visualizations to keep users informed.

### 4. Emergency Contact Services
- Provides contact details for emergency services, including ambulances and healthcare providers.
- Allows direct communication via the platform for faster response times.

### 5. User-Friendly Interface
- Clean, intuitive UI designed for easy navigation under stressful situations.
- Supports multilingual accessibility to cater to diverse audiences.

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/coviheal.git
   cd coviheal
   ```

2. **Install dependencies**
   - Ensure Python and Node.js are installed on your system.
   - Run the following commands:
     ```bash
     pip install -r requirements.txt
     npm install
     ```

3. **Set up the database**
   - Create a MySQL database and configure the connection in the `.env` file.

4. **Run the application**
   - Start the backend server:
     ```bash
     python app.py
     ```
   - Start the frontend:
     ```bash
     npm start
     ```

5. **Access the application**
   - Open your browser and navigate to `http://localhost:3000`.
